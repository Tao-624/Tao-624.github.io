MVP V1.0

最小可行产品

核心目标：用最快速度完成基础功能，验证信息架构

  • 基础页面结构（单页面滚动）
  • 响应式布局（桌面端 + 移动端）
  • 核心信息展示（简介、教育、经历）
  • 基础导航与锚点跳转
  • GitHub Pages 部署上线

产品思考

"先让产品能用，再让产品好用。MVP阶段聚焦核心价值——清晰展示个人信息，其他功能后续迭代。"

V2.0

体验优化版

核心目标：基于用户体验优化交互细节，增加产品质感

  • 暗黑模式（尊重用户偏好）
  • 阅读进度条（减少迷失感）
  • 返回顶部按钮（长页面必备）
  • 加载动画（缓解等待焦虑）
  • 滚动入场动画（增加层次感）
  • 产品迭代模块（展示方法论）

产品思考

"好的产品体验在于细节。暗黑模式不只是功能，更是对用户习惯的尊重。"
